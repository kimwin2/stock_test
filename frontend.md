너는 시니어 프론트엔드 개발자야. 첨부된 주식/테마 정보 대시보드 UI 이미지를 완벽하게 동일한 디자인으로 구현해야 해. 
이 프로젝트는 웹사이트용이 아니라, 모바일 앱과 PC 설치형 프로그램으로 빌드할 거야. 
기술 스택은 [Flutter / React Native 및 React]를 사용할 거야.

아래의 상세 요구사항과 컴포넌트 구조를 바탕으로, 확장 가능하고 깔끔한 코드를 작성해 줘. 
정적 데이터가 아닌, API로 데이터를 받아온다고 가정하고 구조체(Model/Interface)를 먼저 정의한 뒤 UI를 구현해.

### 1. 전체 레이아웃 (Layout)
- **배경색:** 옅은 회색 (앱 전체 배경)
- **Top Header:** - 좌측: 로고 텍스트 ("Stock Premium" - 굵은 청록색 폰트, 주황색 'Premium')
  - 중앙: 현재 날짜 및 시간 ("04-10(금) 21:59")
  - 우측: 아이콘 3개 (뉴스, 히스토리, 캘린더)
  - 하단: 전체 폭을 차지하는 검색창 ("종목, 테마명을 입력하세요.", 우측 돋보기 및 햄버거 메뉴 아이콘)
- **Body:** 2단 격자(Grid) 구조. 모바일 화면에서는 1단, 태블릿/PC 크기에서는 2단으로 반응형 처리할 것.
- **Bottom Footer:** 고정된 하단 뉴스 티커(Ticker). 가로로 자동 스크롤되거나 리스트 형태로 최신 뉴스를 보여줌.

### 2. 테마 카드 컴포넌트 (Theme Card)
- 둥근 모서리(border-radius)를 가진 흰색 박스 카드.
- **카드 헤더:** - 청록색 배경에 흰색 텍스트로 테마명 표시 (예: "광통신", "반도체소부장")
  - 우측에 빨간색 텍스트로 테마 전체 거래대금 표시 (예: "24,680억")
- **서브 헤드라인:** 테마 관련 주요 뉴스 한 줄 요약 텍스트
- **종목 리스트:** 각 카드 안에 4개의 주식 종목 아이템(Stock Item)이 리스트업 됨.

### 3. 주식 종목 아이템 컴포넌트 (Stock Item - ⭐️ 가장 중요)
- 1위 종목은 배경색을 연한 노란색으로 하이라이트 처리.
- **데이터 필드:**
  - 종목명 (좌측 상단)
  - 등락률 (우측 상단, 붉은색 텍스트, 화살표 포함 예: ↑29.93%)
  - 현재가, 거래시간 (좌측 하단)
  - 거래대금 (우측 하단)
- **미니 차트 바 (Range Bar):**
  - 종목 아이템 최하단에 위치한 진행률 바(Progress Bar) 형태의 UI.
  - 회색 얇은 바(오늘의 저가~고가 범위) 위에 빨간색 두꺼운 바(시가~현재가 등락 범위)가 오버레이 됨.
  - 검은색 세로선(틱)으로 기준점(전일 종가 또는 시가)을 표시해야 함.
